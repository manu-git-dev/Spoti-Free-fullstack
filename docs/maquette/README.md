# Maquette Spoti-Free — export statique

Export PNG/JPEG des écrans de la maquette Pencil (`~/Documents/Maquette-Spoti-Free.pen`,
fichier source non versionné, local à la machine où Pencil est installé). Ces images
permettent de consulter le design sur n'importe quelle machine, sans avoir Pencil installé.

À régénérer (depuis une machine avec Pencil et le fichier source) si la maquette évolue.
